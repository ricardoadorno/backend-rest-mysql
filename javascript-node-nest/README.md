## Description

- CRUD

  - Create
  - Read
  - Update
  - Delete

- MySQL Database

  - Connection
  - Queries
  - Tables

- Models
  - User
    - Name
    - Email (Unique)
    - Password (Hashed)
    - Role
  - Comment Post
    - User
    - Title
    - Content
    - Date
    - Likes
    - Dislikes

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
